# config_service

### Environment Variables
Environment variables required to run this service:

- `S3_BUCKET_NAME`: The name of the S3 bucket where the files will be stored.
- `S3_REGION`: The region of the S3 bucket.
- `S3_URL`: The URL of the S3 service.
- `S3_ACCESS_TOKEN`: The access token for S3 authentication.
- `S3_SECRET`: The secret key for S3 authentication.
- `SCYLLA_URL`: The URL for connecting to the Scylla database.
